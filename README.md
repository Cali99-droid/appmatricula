# API MATRICULA

1. Clonar el proyecto
2. `npm install`
3. Clonar el archivo `.env.example` y renombrar a `.env`
4. Cambiar la varibles de entorno
5. Levantar la base de datos

```
docker-compose up -d

```

6. Levantar en desarrollo: `npm run start:dev`
